# API 文档

## 统计研究生数量

### 请求地址

http://apis.ynu.edu.cn/do/api/call/count_yjs

### 请求参数

| 属性   | 类型   | 默认值 | 必填 | 说明       |
| ------ | ------ | ------ | ---- | ---------- |
| lxbz   | String |        |      | 在校标识   |
| xbdm   | String |        |      | 性别代码   |
| pyccm  | String |        |      | 培养层次   |
| xslbdm | String |        |      | 学生类别   |
| nj     | String |        |      | 年级       |
| dszgh  | String |        |      | 导师       |
| sflxs  | String |        |      | 是否留学生 |
| xwlx   | String |        |      | 学位类型   |
| mzdm   | String |        |      | 名族       |
