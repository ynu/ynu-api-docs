### 说明

本文档使用`vuepress`搭建，具体使用可参考 [vuepress](https://v1.vuepress.vuejs.org)

### 文档编辑

``` bash
# 环境搭建
git clone https://github.com/ynu/ynu-api-docs.git
cd ynu-api-docs
yarn

# 启动开发模式
yarn start

# 文档位于 /docs/guide/，格式是markdown
# 配置文件位于 /docs/.vuepress
```

::: warning 注意
请确保你的 Node.js 版本 >= 8.6。
:::
